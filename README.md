# ITZFIZZ Scroll Animation Assignment

A premium scroll-driven animated hero section demonstrating frontend mastery with GSAP and Next.js.

## Tech Stack
- **Framework**: Next.js (App Router) + React
- **Styling**: Tailwind CSS
- **Animations**: GSAP & ScrollTrigger
- **Language**: TypeScript

## Features
- Smooth scroll-driven parallax animations
- Performance optimized using transform properties
- Responsive design tailored for all devices
- High-quality visual assets seamlessly integrated using CSS blend modes

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
